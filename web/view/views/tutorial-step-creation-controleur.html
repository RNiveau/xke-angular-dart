Le contrôleur est l'un des composants fondamentaux du framework AngularJS. C'est une simple fonction qui permet de définir l'état initial de son scope (le modèle, ses attributs) et de lui ajouter du comportement (de nouvelles fonctions).

L'objectif de cette étape est donc la création d'un contrôleur et l'affichage d'un élément de son scope dans la page (en l'occurence les logs) :

  • Dans le fichier workshop-angular/app/js/app.js, définir une function LogCtrl qui prend en paramètre $scope
  • Définir l'attribut logs au sein du $scope et copier/coller le contenu du fichier workshop-angular/data/log-list.json (pour le moment, les logs sont mis en dur dans le contrôleur, par la suite ils seront récupérés depuis un backend)
  • Dans le fichier workshop-angular/app/index.html, ajouter l'attribut ng-controller au div #angular-app, puis afficher le contenu de la variable $scope.logs dans la page
