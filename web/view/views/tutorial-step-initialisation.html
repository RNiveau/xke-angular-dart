L'objectif de cet atelier est de réaliser une application permettant d'afficher et de filtrer des access logs provenant d'un serveur Web à l'aide du framework AngularJS.

Le workshop est décomposé en plusieurs étapes, chacune d'elle est validée par une ou plusieurs assertions exécutées à chaque rafraîchissement de la page. Une fois l'étape passée avec succès, celle-ci est grisée.

La documentation officielle et l'API de référence pourront vous être utiles.

Chaque application Dart contient une méthode main(); qui sera lancée au démarrage de celle-ci.

Pour initialiser l'application AngularDart, il faut définir un module qui s'appelera WorkshopModule et bootstrapper l'application :

  • Dans le fichier xke-angular-dart/web/main.dart, déclarer un module workshop en créant la classe WorkshopModule qui hérite de la classe Module
  • TODO trouver un lien pour ng-app Dans le fichier xke-angular-dart/web/index.html, ajouter la directive ng-app dans l'élément <div id="angular-app" >
  • Dans le fichier xke-angular-dart/web/main.dart, bootstrapper l'application dans la méthode main();
