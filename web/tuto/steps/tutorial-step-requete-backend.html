AngularDart offre la possibilité de requêter un backend de différentes manières, parmi elles, on trouve le service Http.

Cette étape a pour objectif de supprimer les logs déclarées en dur dans le contrôleur et de requêter un backend pour les récupérer.

Pour ce faire :

  • Dans le fichier xke-angular-dart/lib/log_controller.dart, injecter le service Http au sein du contrôleur
  • Requêter les logs à l'URL apache-log.json
  • Utiliser le constructeur nommé Log.fromJsonMap pour créer des objets 'Log' depuis le json récupéré
  • Stocker les logs récupérées au sein d'un attribut 'logs' dans le contrôlleur 'LogController'
