Le two-way data binding> est une fonctionnalité du framework permettant de faire un lien bidirectionnel entre une variable du code Dart et un élément présent dans le DOM. Chaque modification d'un côté ou de l'autre est automatiquement répercutée. En d'autres termes, le framework effectue une synchronisation automatique entre le modèle et la vue.

L'objectif de cette étape est donc la mise en oeuvre de cette fonctionnalité. Pour ce faire :

  • Dans le fichier xke-angular-dart/web/index.html, déclarer l'attribut ng-model (section How Angular does MVC) avec la valeur query au niveau du champ de recherche
  • Afficher le contenu du champ texte (section Mustache syntax and data binding) dans la page
